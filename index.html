How to Contribute

Follow these simple steps to get featured!

  1. Fork the repository to your own GitHub account.
  2. Create a new branch (e.g., add/your-name).
  3. Edit data/contributors.ndjson and add a new line.
  4. Commit your change and open a Pull Request.
  5. Once merged, your profile will appear here automatically.

Full instructions are available in the README.
